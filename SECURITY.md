# Security Policy

## Supported Versions

Security updates are currently provided for the active development branch only.

| Version | Supported |
| --- | --- |
| `main` | Yes |
| older branches | No |

## Reporting a Vulnerability

Please report security issues privately by email:

- `tc3107@proton.me`

When reporting, include:

- A clear description of the issue
- Reproduction steps or proof of concept
- Potential impact
- Any suggested mitigation

Please do not open public GitHub issues for undisclosed vulnerabilities.
